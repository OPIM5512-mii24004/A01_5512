# A01_5512

This repository includes the work for OPIM 5512 Module 1. California housing data is utilized to generate a box plot of house age saved as a png file. To run the script, all required packages must be imported. Proceed by importing the California housing data as a Pandas dataframe. To obtain the box plot, run the matplotlib boxplot function with the "HouseAge" column in the dataframe. The image will be saved in the current working directory.
